## Setup

* No installation required - just open `falling-expressions-menu.html` from menu directory

## Key Features

* Customize game experience
  - difficulty
  - number of lives
  - game mode
  - background color
    
* Move the box in 4 ways
  - arrow keys
  - left and right arrows
  - touch
  - mouse
    
* Responsive
  
* Generate random arithmetical expressions
